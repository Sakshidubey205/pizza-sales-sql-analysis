# 🍕 Pizza Sales SQL Analysis

## 📌 Project Overview

This project analyzes pizza sales data using SQL to uncover insights about customer ordering patterns, popular pizzas, and revenue generation.
The goal is to help businesses make data-driven decisions to improve sales and operational efficiency.

---

## 📊 Dataset

The dataset contains four tables:

* **orders** – order date and time
* **order_details** – pizza quantity per order
* **pizzas** – pizza size and price
* **pizza_types** – pizza name and category

---

## 🔍 Key Insights

* **Total Orders:** 21,350

* **Total Revenue:** $817,860

* **Most Popular Pizza Size:** Large (18,526 orders)

* **Top 5 Most Ordered Pizzas**

  * Classic Deluxe Pizza
  * Barbecue Chicken Pizza
  * Hawaiian Pizza
  * Pepperoni Pizza
  * Thai Chicken Pizza

* **Top Revenue Generating Pizza:**
  Thai Chicken Pizza ($43,434)

* **Average Daily Orders:**
  138 pizzas per day

* **Highest Revenue Category:**
  Classic Pizza (26.9%)

---

## 💼 Business Impact

This analysis helps businesses:

* Identify best-selling pizzas
* Understand peak order hours
* Optimize menu strategy
* Improve inventory planning
* Increase overall revenue
